# 🎓 RGM College Assistant

AI-powered chatbot web application to help students with college-related queries.

---

## 🚀 Features

* 🤖 AI chatbot for answering queries
* 🎤 Voice input (Whisper)
* 🌐 Multi-language support
* 🧠 Smart search using FAISS
* 📍 Location & navigation support
* 🔐 Login/Register system
* 💾 MongoDB database

---

## 🛠️ Tech Stack

* Python
* Flask
* MongoDB
* FAISS
* Sentence Transformers
* Whisper
* Groq API

---

## ⚙️ How to Run

```bash
pip install -r requirements.txt
python app.py
```

Open browser:

```
http://127.0.0.1:5000
```

---

## 🔐 Note

API keys are stored securely using environment variables.
